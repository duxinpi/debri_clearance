An overview description of the CPLEXUtils library.

Introduction

CPLEXUtils contains utility routines that allow a user to specify parameter settings for CPLEX and/or CPOptimizer at run time, rather than having to specify those settings in the code. It also provides a static method for listing in human-readable form a CPOptimizer model.

CPLEXUtils is released under the Eclipse Public License 1.0 (http://www.eclipse.org/legal/epl-v10.html).

Overview

With parameter setters from CPLEXUtils added to a program's code, a user can specify parameter names and values as strings, either as command line arguments or through a file or interactive interface. The setter then locates the named parameter, converts the value to the correct type, and applies the parameter setting to the relevant solver. For more information and a usage example, see the documentation for the cplexutils package.
