This package provides a main program that catalogs and lists the parameters for the linked versions of CPLEX and CPOptimizer.

[Introduction] [Usage]

Introduction

CatalogParameters does not contain code for setting solver parameters. It simply scans the linked solver libraries and outputs an alphabetic list of recognized parameters for each solver. Among other things, this provides a way to recognize when parameter names (e.g., "Display") are ambiguous and how much is needed to disambiguate them.

Usage

To run the program, execute the following:

java -classpath .:<path to CPLEX>/cplex.jar:<path to CPOptimizer>/ILOG.CP.jar:CPLEXUtils.jar program.CatalogParameters

The program requires that the appropriate binary files for CPLEX and CPOptimizer be on the library path. They can be specified either by adding -Djava.library.path=... to the command line or by adding their paths to the LD_LIBRARY_PATH environment variable.
